RaphaëlJS Dots 2.0

Embed with two lines!

Features

Dots has multiple modes of play, from puzzle with custom level support, to elimination or simply endless dot matching.

You can now embed multiple instances of Dots into a single page.

Function Reference

setup
Called once with no argument, ie: GadgetoidDots("dots").setup(), this sets up a Dots game with your specified settings This should be the last thing you call.
set_game_size(dots_x,dots_y)
Set the size of the game in dots. For example .set_game_size(7,7) gives you a 7x7 grid.
set_allowed_moves(horizontal,vertical,diagonal)
Called with three boolean arguments, one to enable horizontal moves, one to enable vertical and one for diagonal. ie: GadgetoidDots("dots").set_allowed_moves(true,true,false)
set_path_col('#FFFFFF')
Set the colour of the join path
set_bg_col('#000000')
Set the background colour of the playing field
set_dot_cols(['#FF0000','#00FF00'])
Set the dot colours. Specify an array of colours
